  • 두번째 인자는 새 창의 이름이다. _self는 스크립트가 실행되는 창을 의미한다.
  • _blank는 새 창을 의미한다.
  • 창에 이름을 붙일 수 있다. open을 재실행 했을 때 동일한 이름의 창이 있다면 그곳으로 문서가 로드된다.
